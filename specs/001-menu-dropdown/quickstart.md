# Quickstart: Menu Dropdown

## Purpose
Add multi-level dropdown support to the Menu component with click-only activation and keyboard navigation.

## How to validate
1. Provide a menu with nested items (3+ levels).
2. Click a menu item to open its submenu.
3. Use Right Arrow to open a child menu, Left Arrow to return to parent.
4. Select an item to trigger its action and close the submenu.
