# Task 2 - Win98 Button Style Evidence

## Modified File
`src/theme/win98/styles/index.scss` - `.cm-button` block (lines 65-117)

## Changes Made

### Default Button (Raised Bevel)
```scss
.cm-button {
    min-width: 75px;       // Spec: 75px min-width
    min-height: 23px;     // Spec: 23px min-height
    padding: 0 12px;
    border: 2px solid;
    border-color: #ffffff #808080 #808080 #ffffff;  // Win98 raised bevel
    border-radius: 0;      // No modern rounding
    background: #c0c0c0;  // Win98 face color rgb(192,192,192)
    box-shadow:
        inset -1px -1px 0 #0a0a0a,   // Darkest inner line (bottom-right)
        inset 1px 1px 0 #dfdfdf;     // Highlight inner line (top-left)
    color: #000000;
    font-size: 11px;
    cursor: pointer;
}
```

### Hover State
```scss
.cm-button:hover:not(:disabled) {
    background: #d4d4d4;  // Slight face brightening
}
```

### Active State (Sunken Bevel - borders reversed)
```scss
.cm-button:active:not(:disabled) {
    border-color: #808080 #ffffff #ffffff #808080;  // Reversed = sunken
    box-shadow:
        inset 1px 1px 0 #0a0a0a,   // Inner darkest now bottom-right
        inset -1px -1px 0 #dfdfdf; // Inner highlight now top-left
    background: #b3b3b3;
}
```

### Focus State (Inner Dotted Outline)
```scss
.cm-button:focus-visible {
    outline: 1px dotted #000000;
    outline-offset: -3px;  // Negative offset = inner/inset focus
}
```

### Primary Variant (Emphasis via bold text)
```scss
.cm-button--primary {
    font-weight: 700;  // Bold for emphasis, same Win98 bevel language
}
```

### Ghost Variant (De-emphasis via lighter face)
```scss
.cm-button--ghost {
    background: #d9d9d9;  // Lighter face for de-emphasis
    box-shadow:
        inset -1px -1px 0 #0a0a0a,
        inset 1px 1px 0 #efefef;  // Softer highlight
}

.cm-button--ghost:active:not(:disabled) {
    background: #cccccc;  // Darker than default active for ghost
}
```

### Disabled State
```scss
.cm-button:disabled {
    color: #808080;           // Gray text
    text-shadow: 1px 1px 0 #ffffff;  // White shadow (Win98 disabled treatment)
    cursor: not-allowed;
}
```
