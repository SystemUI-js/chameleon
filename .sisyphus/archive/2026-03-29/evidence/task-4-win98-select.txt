# Task 4: Win98 Select Styles Optimization - Evidence

## File Modified
- `src/theme/win98/styles/index.scss` (.cm-select block, lines 166-206)

## Changes Applied

### 1. Height: 21px (target from 24px)
- Changed `min-height: 24px` → `min-height: 21px`
- Win98 field controls use 21px height

### 2. Padding: 4px left (reduced from 8px)
- Changed `padding: 0 26px 0 8px` → `padding: 0 16px 0 4px`
- Left padding: 4px (Win98 field standard)
- Right padding: 16px (accommodates arrow area)

### 3. Field-Style Inset Border (Sunken Effect)
- Consolidated to shorthand `border: 2px solid` with `border-color`:
  - `border-color: #808080 #ffffff #ffffff #808080`
  - Top: #808080 (dark), Right: #ffffff (light)
  - Bottom: #ffffff (light), Left: #808080 (dark)
- This creates the proper Win98 sunken/inset border effect

### 4. White Field Face
- Preserved `background-color: #ffffff`
- Removed the old gradient overlay `linear-gradient(180deg, #ffffff 0%, #f0f0f0 100%)`
- Clean white field face

### 5. Win98 Arrow: Pure CSS Using Gradients
- Changed from old multi-layer gradient approach to cleaner dual-gradient:
  - Before: 3-layer gradient (face + two arrow triangles)
  - After: 2-layer gradient (135deg + 225deg black triangles)
- Arrow positioning: `calc(100% - 6px)` and `calc(100% - 11px)` from right edge
- Arrow size: 5px x 5px triangles
- Color: #000000 (pure black, Win98 standard)

### 6. Border Radius: 0
- Added explicit `border-radius: 0` to prevent any modern rounding
- Win98 uses sharp corners

### 7. Focus: Inner Dotted Outline (NOT Blue BG + White Text)
- Added new rule:
  ```scss
  .cm-select:focus {
      outline: 1px dotted #000000;
      outline-offset: -4px;
  }
  ```
- Clearly specified NOT implementing blue background + white text inversion
- Inner dotted outline sits inside the border

### 8. Disabled: Win98 Gray Face + Disabled Text
- `background-color: #c0c0c0` (Win98 gray face)
- Arrow colors change to #808080 (grayed out)
- `color: #808080` (disabled text)
- `cursor: not-allowed`
